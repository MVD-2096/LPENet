## Definition of camouflaged defects
Observation: Such defects can actively or passively blend into the background in terms of visual appearance, thereby evading perception. This visual concealment phenomenon is referred to as camouflage in the computer vision domain. Inspired by this concept, we formally define such defects as **Camouflaged Defects**.\\
![image](https://github.com/MVD-2096/LPENet/blob/main/definition.png)
